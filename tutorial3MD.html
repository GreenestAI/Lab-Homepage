Equilibrium hydrodynamic slip length measurement (how-to)

In order to execute these scripts, you need to download LAMMPS, which is an open source classical molecular dynamics code. You also need a text editor, and a molecular visualisation program, such as VMD. Finally, I use Python to generate the initial configuration.


Step 1 — data file generation

You can download both python file (systemgeneration.py) and generated data file (data.lammps).


Step 2 — reach equilibrium

The first step is to equilibrate the system, by running the following script (in.eq). The script generates a dump as well as a restart file. The dump file can be used using VMD.


Step 3 — measure the slip length

The the following script (in.run) performs a measurement of the slip length and print its value in nanometer. The following lines calculate time correlations of both X-Y forces exerted by water on the top graphene.

variable Nevery equal 10
variable Nrepeat equal 100
variable Nfreq equal ${Nrepeat}*${Nevery}
variable fxt equal fcm(gr1,x)
variable fyt equal fcm(gr1,y)
fix acor all ave/correlate ${Nevery} ${Nrepeat} ${Nfreq} v_fxt v_fyt type auto file correlate.dat ave running

Then, correlated values are integrated and the numerical factor "scale" converts data in meter in agreement with the Green-Kubo formulation.

variable v11 equal trap(f_acor[3])*${scale}
variable v22 equal trap(f_acor[4])*${scale}
timestep ${dt}
run 100000
variable b equal 1e9*(v_v11+v_v22)/2.0
print "slip length: ${b} [nm]"
