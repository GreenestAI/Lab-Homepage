LAMMPS: writting a loop (how-to)


Example 1

LAMMPS requires a few things to be define in any code, such as the simulation box, so to test the "loop" script given below, just insert the following lines in the script.

region simbox block 0 1 0 1 0 1
create_box 1 simbox
pair_style lj/cut 1
pair_coeff * * 1 1
mass * 1

The following lines perform a loop over the counter "ijk", from 1 to 10. The codes also generates a new output at avery iteration. Finally a breaking condition allows to write a criteria for breaking the loop sooner if needed.

variable nb equal 10 # desired number of run
label loop
variable ijk loop ${nb} # counter variable
variable var equal 1*${ijk} # example of measurement
fix myavetime all ave/time 1000 1 1000 v_var file output${ijk}.dat # example of ave/time
print "${ijk}" # terminal printing of the counter value
run 1000
if "${ijk}>${nb}" then "jump input break" # breaking of the loop - can be usefull
unfix myavetime
next ijk
jump input loop
label break
variable ijk delete
